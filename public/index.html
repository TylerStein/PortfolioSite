    Alexa, Run! Was created within 48 hours for the 2018 Global Game Jam under the theme of "What home means to you". As a group of four, and a first jam for most of us, it was a great learning experience and a lot of fun all around. The game has since had its code cleaned up and assets optomized for a stable web release, the files for which can be found on my GitHub.
